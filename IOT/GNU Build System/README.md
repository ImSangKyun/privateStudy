GNU Build System
============================
빌드 자동화 툴 개발을 위한 연습으로
GNU Build System의 문법 공부 및 정리

---------------------------------------------------
### GNU Build System을 사용해야 하는 이유

1. 빌드를 자동화 해야함
	리눅스 환경에서 빌드할 파일이 다수 있고, 이를 링커로 묶어야 할 때 한줄한줄 빌드할 수 없고, 또한 쉘 스크립트로 자동화도 가능은 하지만, GNU build system의 "Incremental build"를 할 수 없다. (변경사항이 있는 부분만 새로 빌드하는 방식)
	또한 리눅스에서 컴파일 할 때 gcc -o main.c main.c 와 같은 명령어를 입력할 경우 대참사가 일어난다. (물론 Git을 사용하면 이전으로 되돌릴수는 있으나, 그동안 들인 시간은....)
	따라서, 빌드는 자동화하는것이 상당히 이롭다.

2. 배우기만 하면 생각보다 원리가 간편하다
	물론 완전 세부적으로 사용하는 문법은 좀 난이도가 있을 수도 있긴한데, 할만한편이다
